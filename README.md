## 📦 Installation

To set up this project on your local machine, follow these steps:

```bash
# Clone the repository
git clone https://github.com/kyreki/awesome-project.git

# Navigate into the project directory
cd awesome-project

# Install dependencies
npm install

# Start the project
npm start
```
